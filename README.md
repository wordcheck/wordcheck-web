# 워드체크(WordCheck)

<i>영단어 기록, 시험, 오답노트 서비스</i>

<br>

## 1. 진행 기간

진행 기간 : 12월 23일 ~ 1월 31일

## 2. 기능소개

공부할 단어를 그룹별로 구분하여 나만의 단어장을 만들 수 있습니다.  
무작정 단어만 보면서 외우는 것이 아닌 다양한 퀴즈를 통해 재미있고 자연스럽게 영어단어를 외울 수 있습니다.  
TTS를 통해 단어의 정확한 발음을 들으며 공부할 수 있습니다.
